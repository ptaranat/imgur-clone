# shopify-challenge

# Installation

`sudo npm install -g @vue/cli`  
`sudo npm update -g @vue/cli`  
`vue create frontend`  
In /backend:  
`npm install --save express`

# Terraform

Install terraform from zip
`tf init`  
Git ignore the .terraform cache.  
Set up AWS access keys (Account->My Security Credentials)
`tf plan`  
`tf apply`
